# Debates

Register of design decisions for Dead Light Framework that are under debate, currently being decided, or recently resolved.

## Why a separate folder

The `*-for-debate.md` documents one level up are *working drafts* of large artifacts (the glossary, a phase definition). The files here are *specific arguments* — narrow questions raised against those drafts, analyzed from multiple angles, and resolved into a decision.

Keeping them separate means:
- The drafts stay readable as drafts.
- Each decision has its own URL to cite later.
- Decisions accumulate as a visible audit trail of how the framework converged.

## Status legend

| Status | Meaning |
|---|---|
| **open** | Question raised, analysis still pending. |
| **recommended** | Analysis complete with a recommendation; awaiting user decision. |
| **decided** | Closed. Outcome has been applied to the relevant draft / doc. |
| **superseded** | Replaced by a later debate. Linked forward. |
